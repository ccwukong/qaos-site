# qaos Landing Page Bundle

This folder is ready to copy into your new landing-page repo.

## Files
- `index.html` — one-page landing site with SEO, Open Graph, Twitter, JSON-LD
- `styles.css` — page styling
- `robots.txt` — crawler rules
- `sitemap.xml` — sitemap
- `manifest.webmanifest` — PWA metadata
- `404.html` — fallback page
- `assets/robot-logo.svg` — robot logo/fallback favicon

## Before Publishing
Replace `https://YOUR_DOMAIN` in:
- `index.html`
- `robots.txt`
- `sitemap.xml`

## Publish on GitHub Pages
1. Create new public repo (e.g. `qaos-site`).
2. Copy all files from this folder to repo root.
3. Push to `main` branch.
4. In GitHub: `Settings -> Pages`.
5. Source: `Deploy from a branch`, Branch: `main`, Folder: `/root`.

Your site should be live in 1-2 minutes.
